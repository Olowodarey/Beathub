# Beathub

A monorepo project containing both frontend and backend applications.

## Structure

- `frontend/` - Next.js frontend application
- `backend/` - NestJS backend application

## Getting Started

### Frontend

```bash
cd frontend
pnpm install
pnpm dev
```

### Backend

```bash
cd backend
pnpm install
pnpm start:dev
```

## Development

This is a monorepo managed with pnpm workspaces.
